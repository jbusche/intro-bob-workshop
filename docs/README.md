# Introduction to IBM Bob Workshops

## Accelerate Your Work with Agentic AI

Welcome to the IBM Bob Workshop series! These hands-on workshops demonstrate how IBM Bob, an agentic AI assistant, can transform your work through practical exercises. Whether you're a student exploring AI for the first time or an IBM professional looking to accelerate your development workflow, we have a workshop designed for you.

### What is Bob?

Bob is an **agentic AI assistant** that goes beyond traditional chatbots. While regular AI assistants answer questions and provide information, Bob can:

- **Take autonomous actions** to complete tasks
- **Create and modify files** independently
- **Break down complex problems** into executable steps
- **Use tools** to read, write, and analyze code
- **Iterate and refine** based on results and feedback

## What You'll Learn

### Core Concepts
- Understanding agentic AI vs. traditional chatbots
- How AI agents use tools to accomplish tasks
- Writing effective prompts for AI assistants
- Iterating and refining AI-generated outputs

### Practical Skills
- Creating interactive web applications
- Analyzing and improving existing code
- Generating documentation and reports
- Automating repetitive tasks
- Working collaboratively with AI

### Real-World Applications
- Rapid prototyping and development
- Code review and quality assessment

## Compatibility

These workshops have been tested with:

- **IBM Bob**: Latest version with VS Code extension
- **Operating Systems**: macOS, Windows 11, Linux
- **Browsers**: Chrome, Firefox, Safari, Edge (for viewing created projects)

- **IBM Bob**: Agentic AI assistant with autonomous tool usage
